# Universal Device Monitor Tool

Python tool that detects:

✔ USB devices
✔ Ethernet connections
✔ COM Ports
✔ Storage devices
✔ Network interfaces

## Technologies Used

- Python
- Socket Programming
- WMI
- psutil

## Features

- Real-time monitoring
- Auto device detection
- Network discovery

## How to Run

Install requirements:

pip install psutil wmi pyserial

Run:

python monitor.py
